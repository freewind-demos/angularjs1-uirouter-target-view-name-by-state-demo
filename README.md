AngularJS1 "@uirouter" Target View Name by State Demo
=====================================================

默认情况下，child state中`views`中给出的view name是parent state产生的。

有时候，我们需要打破这个约定，让child state自己选择target到一个不是由parent state产生的uiview中。

```
npm install
npm run demo
```
